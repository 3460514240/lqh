# lqh

## 项目简介

本仓库用于存放基于 RDK X5、YOLO、ROS、ESP32 和 Orbbec 深度相机相关的项目文件，主要包含电机驱动、目标检测模型训练、模型推理部署以及机器人视觉感知相关内容。

## 目录说明

- 推理：模型推理及部署相关文件
- yolo：YOLO 模型训练、数据配置及检测相关文件
- esp32-s3：电机驱动源码

## 技术栈

- Python
- YOLOv8
- ROS
- RDK X5
- Orbbec 深度相机
- Linux
- Docker
- ONNX
- can
- sbus

## 主要功能

- YOLO 目标检测模型训练
- 模型推理与结果验证
- 深度相机数据接入
- ROS 环境下视觉感知功能调试

## 改进方向
- 目前尚未实现上位机与下位机之间的通信，后续计划基于 micro-ROS 实现 ROS2 上位机与 ESP32 下位机的数据交互。
- 目前还是sbus协议,使用遥控器控制

## 使用说明
- Orbbec 深度相机的代码可以在官方下载，作用是topic，我们的推理代码去subscribe（订阅）
- 练的模型是pt，不能直接部署在rdk x5上，要在docker容器里面将pt->onnx
- 我遇到的问题：在模型量化的时候，图片校验不要开自动，先改图片格式，这个过程不要归一化，在yaml配置文件里面归一化，我是这样解决的
