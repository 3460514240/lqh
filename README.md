# lqh

## 项目简介

本仓库用于存放基于 RDK X5、YOLO、ROS 和 Orbbec 深度相机相关的项目文件，主要包含目标检测模型训练、模型推理部署以及机器人视觉感知相关内容。

## 目录说明

- 推理：模型推理及部署相关文件
- yolo：YOLO 模型训练、数据配置及检测相关文件

## 技术栈

- Python
- YOLOv8
- ROS
- RDK X5
- Orbbec 深度相机
- Linux
- Docker
- ONNX

## 主要功能

- YOLO 目标检测模型训练
- 模型推理与结果验证
- 深度相机数据接入
- ROS 环境下视觉感知功能调试
